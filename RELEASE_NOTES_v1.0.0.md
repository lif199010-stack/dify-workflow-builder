# dify-workflow-builder v1.0.0

## Highlights

- First public release of `dify-workflow-builder`
- Business-to-Dify workflow design and YAML generation
- Review / refactor / validate flow for existing Dify YAML
- Nanny-mode guided workflow building
- URL-driven entry into existing Dify deployments/apps
- Built-in templates for customer service, API orchestration, RAG, OCR, agent, iteration, aggregation, classification, knowledge retrieval, and assigner flows
- Dify bug troubleshooting runbook included

## Positioning

This project is designed as a practical Agent Skill for users who want to move from business requirements to importable Dify workflows, instead of hand-writing raw YAML from scratch.
